# HuaweiRecon
A script capable of logging into Huawei routers (HG8145V5, EG8145V5, HG8546M, HS8546V5, HG8121H, etc...), allowing the extraction of information about connected users, including device details, IP addresses, MAC addresses, and the network name (SSID).
